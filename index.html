Surgimento do HTML

HTML (Hypertext Markup Language) é a linguagem de marcação padrão para a criação e organização de páginas da web.
é usado para estruturar o conteúdo de uma página, definir a semântica dos elementos
e fornecer instruções sobre como a página deve ser exibida em um navegador.

Ao longo dos anos, o HTML evoluiu e novas versões foram lançadas, cada uma com recursos mais avançados.
A última versão estável é o HTML5, que foi lançado em 2014 e introduziu muitos recursos avançados,
como áudio e vídeo embutidos, elementos de formulário aprimorados e suporte para aplicativos web.

Como fazer um parágrafo?

Em HTML, um parágrafo é criado usando a tag "p". Você simplesmente envolve o texto que deseja ser exibido como um parágrafo com essa tag.
O HTML inicialmente continha apenas um conjunto limitado de tags, como p para parágrafos, "a" para links e "h1" a "h6" para cabeçalhos.
À medida que a web evoluiu e se tornou mais popular, o HTML foi expandido com novas tags e recursos em versões posteriores.

Elementos de formatação de texto

Em HTML, existem vários elementos que podem ser usados para formatar o texto em uma página da web. Aqui estão alguns dos principais elementos de formatação de texto em HTML: Títulos e cabeçalhos: Os elementos "h1" a "h6" são usados para criar títulos e cabeçalhos. O "h1" é o mais importante e usado e representa o título principal, enquanto "h6" é o menos usado.
